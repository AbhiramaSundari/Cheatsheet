MY Git Cheatsheet

This website describes basic git commands that i need to know to manage my code in github

All git commands are executed in the command line

FOllowing is the description on how to use the git command

Git clone

With the gitclone i can clone a git repository into my computer's folder structure

Example git clone: https://github.com/AbhiramaSundari/Cheatsheet

Git Status

With this command i can check the status of the folder. It colors the unsaved folders into red and saved folders to right.

Git add .

With this command , i can add all the changed files

git commit -m

With this command, i can commit the changed files with a message. The message should be inserted inside the quotes

Git Push

With this command, i can push my github files to the respository
